Tip:

  • Use CTRL+V to paste a image (from PrintScreen or other programm ).
  • Use CTRL+C to copy current image.
  • Use CTRL+S to save current image.
  • Use CTRL+D to grab a PrintScreen
  • Use ESC to reset the current image.
  • Use + or - to zoom image
  • To move crop selection use Mouse + CTRL or ALTGr
